# 快速安裝指南

## 步驟 1：載入擴充功能

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入：`chrome://extensions/`
3. 點擊右上角的「開發人員模式」開關（確保已開啟）
4. 點擊「載入未封裝項目」按鈕
5. 選擇 `course_registration` 資料夾
6. 看到擴充功能出現在列表中即表示安裝成功

## 步驟 2：載入課程資料

### 方法 A：自動載入（推薦）

1. 訪問 https://timetable.nycu.edu.tw/
2. 擴充功能會自動開始抓取課程資料
3. 等待 2-5 分鐘（取決於網路速度）
4. 看到「已載入 XXX 筆課程資料」通知即表示完成

### 方法 B：使用測試資料（快速測試）

1. 訪問 https://timetable.nycu.edu.tw/
2. 按下 `Ctrl + Shift + L`
3. 立即載入 3 筆示範課程資料

## 步驟 3：開始搜尋

1. 點擊瀏覽器工具列的擴充功能圖示（拼圖圖示）
2. 在彈出視窗的搜尋框輸入課程名稱或代碼
3. 按 Enter 或點擊「搜尋」按鈕
4. 查看搜尋結果

## 搜尋技巧

- 輸入課程名稱：`微積分`、`計算機概論`
- 輸入課程代碼：`MATH101`、`CS101`
- 輸入系所名稱：`資訊工程`、`應用數學`
- 輸入學院名稱：`工學院`、`電機學院`
- 輸入教師姓名：`王大明`

## 故障排除

### 問題：沒有看到課程資料

**解決方法：**
1. 確認已訪問 https://timetable.nycu.edu.tw/
2. 按 F12 開啟開發者工具，查看 Console 訊息
3. 使用 `Ctrl + Shift + L` 載入測試資料先行測試
4. 點擊「重新載入課程資料」按鈕清除舊資料

### 問題：搜尋沒有結果

**解決方法：**
1. 檢查是否已成功載入課程資料
2. 嘗試更簡短的關鍵字（如只輸入「微積」而非「微積分一」）
3. 檢查拼寫是否正確

### 問題：抓取資料失敗

**解決方法：**
1. 檢查網路連線
2. 確認可以正常訪問 timetable.nycu.edu.tw
3. 重新整理頁面後再試一次
4. 查看開發者工具 Console 的錯誤訊息

## 更新資料

課程資料會自動保存 24 小時。如需手動更新：

1. 點擊擴充功能圖示
2. 點擊「重新載入課程資料」按鈕
3. 重新訪問 https://timetable.nycu.edu.tw/

## 卸載

1. 訪問 `chrome://extensions/`
2. 找到「NYCU 課程搜尋助手」
3. 點擊「移除」按鈕
