======================================
Chrome Web Store 擴充功能說明
======================================

【簡短描述】（最多 132 字元）

快速搜尋 NYCU 課程！支援課程名稱、代碼、系所、學院、教師、時間、教室搜尋，顯示完整選課路徑。

--------------------------------------

【詳細描述】

NYCU 課程搜尋助手

專為陽明交通大學（NYCU）學生設計的課程搜尋工具，讓選課變得更簡單！

========== 主要功能 ==========

▼ 強大的搜尋功能
・支援課程名稱、課程代碼、教師姓名搜尋
・支援系所、學院、課程類型搜尋
・支援上課時間、教室搜尋
・智能簡稱識別（如「邏設」自動匹配「邏輯設計」）
・多條件搜尋（用空格分隔，例如：「微積分 王禹超」）

▼ 完整選課路徑顯示
・顯示課程的所有選課入口
・清楚標示：課程類型 / 類別 / 學院 / 系所
・一門課有多個路徑時自動標號顯示

▼ 智能資料管理
・自動從 NYCU 官方 API 抓取最新課程資料
・本地儲存 7 天，無需重複載入
・支援手動更新資料
・即時顯示資料狀態和更新時間

▼ 便捷的使用體驗
・側邊欄設計，不影響原網頁瀏覽
・在選課網站上自動啟用
・任何網站都可開啟搜尋
・即時搜尋結果顯示

========== 使用方式 ==========

【第一次使用】
1. 安裝擴充功能
2. 訪問 https://timetable.nycu.edu.tw/
3. 等待約 5 分鐘自動載入課程資料
4. 開始搜尋！

【搜尋範例】
・課程名稱：「微積分」
・課程簡稱：「邏設」「線代」「計組」
・課程代碼：「DCP1192」
・教師：「王禹超」
・系所：「資工系」
・學院：「電機學院」
・時間：「週一」
・教室：「EC015」
・多條件：「邏設 一年級」「資工 演算法」

【快速開啟】
・點擊瀏覽器工具列的擴充功能圖示
・或在 timetable.nycu.edu.tw / cos.nycu.edu.tw 自動顯示側邊欄

========== 隱私保護 ==========

・所有資料儲存在本地瀏覽器
・不收集任何個人資訊
・僅訪問 NYCU 官方課程 API
・開源透明

========== 技術特色 ==========

・使用 NYCU 官方課程時間表 API
・Chrome Extension Manifest V3
・側邊欄（Side Panel）技術
・高效的本地搜尋演算法

========== 資料來源 ==========

所有課程資料來自 NYCU 課程時間表系統官方 API (https://timetable.nycu.edu.tw/)，確保資訊準確即時。

========== 注意事項 ==========

・首次使用需載入課程資料（約 5 分鐘）
・資料每 7 天自動更新一次
・目前涵蓋所有課程類型（學士班、碩士班、博士班等）
・建議定期更新以獲取最新課程資訊
