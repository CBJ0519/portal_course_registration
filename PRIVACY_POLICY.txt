【3. 單一用途說明 (Single Purpose)】

本擴充功能的單一用途是：為陽明交通大學學生提供課程搜尋功能。

使用者可以透過課程名稱、代碼、系所、教師等關鍵字，快速搜尋 NYCU 課程資料，並查看課程的完整選課路徑、上課時間、地點等資訊。所有功能僅用於幫助學生更有效率地搜尋和查詢課程資訊。

--------------------------------------

【4. 權限說明 (Permission Justifications)】

▼ activeTab 權限
理由：
用於在使用者點擊擴充功能圖示時，獲取當前分頁資訊以開啟側邊欄。此權限確保側邊欄能在正確的視窗中開啟，不會收集或存取分頁內容。

▼ sidePanel 權限
理由：
用於提供側邊欄介面，讓使用者在瀏覽選課網站時可以隨時搜尋課程，而不影響原網頁操作。側邊欄僅顯示課程搜尋功能，不會存取或修改網頁內容。

▼ storage 權限
理由：
用於在瀏覽器本地儲存從 NYCU 官方 API 抓取的課程資料（課程名稱、代碼、時間、教室等公開資訊），避免每次使用都需要重新下載數千筆課程資料。所有資料僅儲存在使用者的本地瀏覽器中，不會上傳到任何伺服器。資料每 7 天自動更新一次。

▼ 遠端程式碼 (Remote Code)
理由：
本擴充功能不使用任何遠端程式碼。所有程式碼都包含在擴充功能套件中，不會從外部載入或執行任何腳本。

如果系統偵測到遠端程式碼使用，可能是因為：
- 擴充功能從 NYCU 官方 API (https://timetable.nycu.edu.tw/) 獲取課程「資料」（JSON 格式），這是純資料而非可執行程式碼
- 所有資料處理邏輯都在本地執行

▼ 網站存取權限 (Host Permissions)
網站：https://timetable.nycu.edu.tw/*

理由：
用於從 NYCU 官方課程時間表系統的 API 獲取課程資料。擴充功能需要存取此網站的 API endpoints 來下載公開的課程資訊（課程名稱、代碼、時間、教室、教師等）。

存取權限僅用於：
1. 呼叫 NYCU 官方 API 獲取課程資料
2. 在 timetable.nycu.edu.tw 網站上自動啟用側邊欄，方便使用者搜尋

不會收集任何個人資訊或瀏覽記錄。

--------------------------------------

【5. 資料使用情形 (Data Usage)】

▼ 收集的資料
本擴充功能不收集任何使用者個人資料。

擴充功能僅在使用者的本地瀏覽器中儲存：
- 從 NYCU 官方 API 獲取的公開課程資料
- 資料最後更新時間

▼ 資料處理方式
- 所有課程資料僅儲存在使用者的本地瀏覽器 (Chrome Storage API)
- 資料不會上傳到任何遠端伺服器
- 資料不會與第三方分享
- 資料每 7 天自動更新

▼ 隱私權政策
- 不收集個人識別資訊
- 不追蹤使用者行為
- 不使用 cookies 或分析工具
- 不顯示廣告
- 開源透明
